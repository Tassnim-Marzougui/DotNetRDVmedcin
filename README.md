Ce projet est une application web de réservation de rendez-vous médicaux pour une clinique, développée avec .NET. Elle permet aux patients de consulter les médecins disponibles et de réserver un rendez-vous en ligne de manière simple et rapide. L’application intègre un chatbot intelligent pour assister les utilisateurs, répondre aux questions fréquentes et guider le patient durant le processus de réservation. Un système de notification par email est mis en place afin de confirmer automatiquement les rendez-vous et informer les patients et les médecins. Un dashboard interactif permet à l’administration de gérer les médecins, les patients et les rendez-vous, offrant ainsi une solution moderne, efficace et digitale pour la gestion des services médicaux.
